## Sobre este Repositorio :book:
Este repositorio aloja todas las soluciones, ejercicios y proyectos prácticos desarrollados durante el curso **Full Stack Open** impartido por la **Universidad de Helsinki**.

El objetivo de este repositorio es profundizar en el desarrollo web moderno, yendo más allá de los fundamentos para dominar el stack MERN (MongoDB, Express, React, Node.js), así como conceptos avanzados de **CI/CD**, **Containerización**, **GraphQL** y **TypeScript**. Aquí encontrarás desde lecciones de repaso hasta arquitecturas backend completas y tests automatizados.

## Stack de Aprendizaje :warning:
A lo largo de este curso, estoy implementando y fortaleciendo mis habilidades en las siguientes tecnologías:

* **Frontend:** React (Hooks, Redux, React Router), Axios, Vite.
* **Backend:** Node.js y Express para la creación de APIs RESTful robustas.
* **Base de Datos:** MongoDB (NoSQL) con Mongoose.
* **Testing:** Jest, Vitest, Cypress y Playwright para pruebas End-to-End (E2E).
* **Tipado:** TypeScript para un desarrollo más seguro y escalable.
* **Gestión de Estado:** Redux Toolkit y React Query.
* **Otros:** GraphQL (Apollo Server/Client) y Contenedores Docker.

## Pasos de Ejecución :checkered_flag:

Dado que este es un monorepositorio con múltiples proyectos pequeños, cada carpeta suele tener su propio `package.json`.

1.  **Clona el repositorio**
    ```bash
    git clone [https://github.com/TatoNaranjo/fullstackopen.git](https://github.com/TatoNaranjo/fullstackopen.git)
    cd fullstackopen
    ```

2.  **Navega a la parte deseada**
    Por ejemplo, para ver el backend de la parte 3:
    ```bash
    cd part3
    ```

3.  **Instala las dependencias y corre el proyecto**
    ```bash
    npm install
    npm run dev
    ```
